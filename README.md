# 🎨 Color Flipper

A simple JavaScript project that changes the background color randomly when clicking a button.

## 📌 Features

✅ Random background color generation  
✅ Simple UI with a single button  
✅ Easy to understand JavaScript logic

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
